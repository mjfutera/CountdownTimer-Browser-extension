Countdown Timer: Never Miss a Thing

Features

• Create custom countdown timers
• Set any time for each timer, from 1 minute to 365 days
• Choose whether you want the timer to repeat
• Add a title and description to each timer
• Start and stop timers — they start automatically when saved and stop when the time runs out
• Receive browser notifications with a bell sound alert
• Optionally open a specific website when the time runs out
• Reorder your timers with drag up/down
• Timezone support — set events in any timezone
• All timers visible on your New Tab page
